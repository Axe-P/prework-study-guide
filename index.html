HTML

  • HTML is like the backbone or frame of the webpage.
  • Elements and Tags act the same, but Tags are located inside an Element.
  • <meta> Element store metadata such as the character set, description, and viewport.
  • <br> Element adds a horizontal break between two bodies of code or text/image being displayed.

CSS

  • Using * as the selector will apply rule to all elements.
  • There are three different types of CSS, Internal, External, and Inline.
  • It is possible to embed an entire CSS sheet into the head of an HTML file.
  • External CSS is usually prefered because you can modify the rules in another file.

Git

  • Bash and GitBash are essentially the same.
  • Can interact with your machine locally and on the "cloud"
  • The "git branch" command will tell you which branch you are on and what branches are on the repository.
  • The "git checkout -b 'branch-name'" creates a new branch and switches to it

JavaScript

  • Arrays are usefull tools for storing data, they can store strings, numbers, and boolean values.
  • Unlike CSS files which are usually located in the head of the html code. JavaScript files are located in the body
